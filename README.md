# Проект Mesto
1. Описание: одностраничный сайт "Место". Использована адаптивная, резиновая верстка, флексбокс-вёрстка, грид-верстка, позиционирование и продвинутая семантика языка HTML. Часть логики написана на JavaScript, в частности реализован диыалоговые окна из макета — «Редактировать профиль», «Добавить карточку», отрытие карточки при нажатии . Файловая структура проекта оформлена по Nested БЭМ.
2. Планы по доработке проекта: внести правки от ревьюера.
